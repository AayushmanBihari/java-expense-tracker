# Java Expense Tracker

A console-based Expense Tracker application built using Core Java and DSA concepts.

## Features
- Add and view expenses
- Month-wise sorting of expenses
- Binary search for expense by month
- Category-wise expense analysis
- File handling for data persistence

## Concepts Used
- ArrayList
- Sorting (Comparator)
- Binary Search
- HashMap
- File Handling
- Object-Oriented Programming

## How to Run
```bash
javac ExpenseTracker.java
java ExpenseTracker
